# To Do List Vue.js

## Giới thiệu
Ứng dụng này cho phép người dùng thêm, xóa và đánh dấu các công việc đã hoàn thành. Được xây dựng bằng Vue.js, ứng dụng có thiết kế đơn giản và dễ sử dụng.

## Tính năng
- Thêm công việc mới
- Xóa công việc
- Đánh dấu công việc đã hoàn thành
- Lưu và tải danh sách công việc từ LocalStorage

## Cài đặt
1. Clone repo này về máy của bạn:
   ```bash
   git clone https://github.com/username/to-do-list-vuejs.git
   ```
2. Chuyển vào thư mục dự án:
   ```bash
   cd to-do-list-vuejs
   ```
3. Cài đặt các gói cần thiết:
   ```bash
   npm install
   ```
4. Khởi động ứng dụng:
   ```bash
   npm run serve
   ```

## Công nghệ
- Vue.js
- HTML/CSS

## Các ghi chú khác
- Ứng dụng này được xây dựng hướng đến việc học tập và thực hành Vue.js cho các lập trình viên mới.
